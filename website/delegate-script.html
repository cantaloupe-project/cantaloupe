Delegate Script

The delegate script mechanism enables the use of custom Ruby functions as "hooks" to provide dynamic information back to the image server. A truly dynamic and customized image server can be created with minimal lines of code.

A sample delegate script, delegates.rb.sample, is included in the distribution. This should be copied to delegates.rb and referenced from the delegate_script configuration option, like: 

# Option 1: absolute pathname
delegate_script = /path/to/delegates.rb

# Option 2: filename only; will be expected to be located in the
# same folder as the config file, or the current working directory
delegate_script = delegates.rb

The delegate script mechanism is optional and does not have to be used at all. In that case, the delegate_script option can be commented out or left blank.

Rules

While the arguments and return types of each function will vary, the script itself must contain a Cantaloupe module within which all delegate functions are contained.

Inside a function, anything goes—it will be interpreted by a JRuby interpreter built into Cantaloupe, so you can even use any (non-platform-native) gems that you have installed with gem install.

Because it will be called upon each image request, though, it should be written with efficiency in mind.

Scripts will be re-read on each request, so they can be edited without restarting.

Note: typically, neither function arguments nor return values are sanitized or validated. Be very careful to write defensive, injection-aware code.

Example

Here is an example of a script used by FilesystemResolver that performs a Solr query to return a pathname based on an identifier. The documentation in that section describes the contract that this function must abide by: its name, arguments, and return value.

require 'cgi'
require 'net/http'

module Cantaloupe
  def self.get_pathname(identifier)
    uri = 'http://localhost:8983/solr/collection1/select?q=' +
        CGI.escape('id:"' + identifier + '"') +
        '&fl=pathname_si&wt=ruby'
    uri = URI.parse(uri)

    http = Net::HTTP.new(uri.host, uri.port)
    request = Net::HTTP::Get.new(uri.request_uri)
    response = http.request(request)
    return nil if response.code.to_i >= 400

    results = eval(response.body)['response']['docs']
    results.any? ? results.first['pathname_si'] : nil
  end
end
