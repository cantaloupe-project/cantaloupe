Endpoints

IIIF Image API 1.x

Compliance

Cantaloupe implements version 1.1 of the Image API, for compatibility with all 1.x clients. Compliance level is dynamically computed and declared on a per-processor basis. Most processors are "Level 2"-compliant.

Location

This endpoint is available at /iiif/1.

Enabling/Disabling

This endpoint can be enabled or disabled by setting endpoint.iiif.1.enabled to true or false in the configuration file.

IIIF Image API 2.x
